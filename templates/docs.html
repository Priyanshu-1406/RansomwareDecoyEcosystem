üìå Overview

The Ransomware Decoy Ecosystem is a controlled environment designed to detect ransomware activity and analyze its behavior in a secure manner. The system functions by deploying decoy files and monitoring for modification to identify potential ransomware attacks. When suspicious activity is detected, the system takes an instant backup of affected files, logs the activity for future forensics, disbles wifi to prevent futher spreading, and alerts the user via a desktop/email notification.

This project is designed to be Windows-based and implemented using Flask for the backend with an HTML & CSS frontend for user interaction.

üõ† Features

  • Decoy File Deployment: Creates multiple decoy files to attract ransomware attacks.
  • Real-Time Monitoring: Constantly watches for suspicious file access and encryption activity.
  • Backup System: Takes an time to time backup so that user has access to data even after the attack
  • User Notification: Sends desktop alerts and alert email when potential modification activity is found.
  • Web-Based Interface: Provides a user-friendly web dashboard for monitoring and control.
